How to use

  • Enter digits 1–9. Leave cells empty for unknown values.
  • Use “Load Grid” to paste a full board quickly.
  • Press “Solve” to solve the current puzzle.
  • Press “Check” to validate the current entries.
  • Use “Reset” to restore the default puzzle.
  • Use “Clear” to empty the board.
  • Select a difficulty and press “New Game” to generate a puzzle.
  • Fill the grid. Wrong entries cost a life.
  • You have up to 3 hints.
  • Finish the puzzle to stop the clock.
